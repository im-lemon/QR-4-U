# QR-4-U

 ---

- QR-4-U is a small Python CLI-tool that generates QR-codes from the link provided, it is simple and easy to understand, since it uses the Python qrcode library.


- The tool stores output in the folder "output-qr-4-u". This where the tool stores your QR Code.
